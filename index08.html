Apresentação de mim

Olá chamo-me Joaquim Eliseu Menianga, estudante da ciencia da computação e venho partilhar as minhas rotinas a respeito de estudo da tecnologia

Olá! Sou um entusiasta de tecnologia e desenvolvedor aspirante de Luanda, Angola, apaixonado por programação e desenvolvimento web. Atualmente, domino fundamentos de Python, HTML, CSS, JavaScript e SQL, com experiência em projetos pessoais usando Flask, Django e SQLAlchemy para criar APIs RESTful e interfaces web.

Meu foco é evoluir para níveis avançados: pretendo dominar Docker para containerização, Git e GitHub para controle de versão eficiente, FastAPI para APIs robustas, bibliotecas de análise de dados como Pandas e Matplotlib, além de conceitos de design de software, testes unitários e UI/UX básicos. Meu objetivo é construir projetos completos, como apps web com backend escalável e análise de dados, enquanto melhoro minha produtividade com rotinas de estudo estruturadas.
